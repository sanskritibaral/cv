Academic Projects

3D Modelling of KU Admin Block

Developed a detailed 3D model of Kathmandu University Admin Block using spatial and visualization techniques for structural representation and analysis.

Suitability Analysis for Dumpsite in Pokhara (MCDA)

Conducted Multi-Criteria Decision Analysis (MCDA) using GIS techniques to identify suitable dumpsite locations. Performed weighted overlay analysis using spatial factors including land use, road networks, settlements, slope, and water bodies.

GIS-Based Routing of Transmission Line

Developed optimal transmission line routes using Least Cost Path Analysis in GIS (Tamakoshi to Kalinchowk). Utilized DEM, slope, LULC, and spatial constraints for route modeling.

Survey Camp

Conducted practical surveying works including leveling, traversing, triangulation, hydropower survey, bridge survey, plane table survey, and route survey using professional instruments.
